重寫了一次簡單的網誌系統，知道client和server如何互動，ＣＳＳ以及資料庫的建置會再找時間完成。

1.可以新增貼文

2.可以閱讀貼文

3.可以再新增貼文的頁面點選重置以及取消

4.主頁面可看到目前有幾筆貼文
